## 📌 O projektu

</br>

Osobní portfolio vytvořené s důrazem na **responzivní design** a přístup **Mobile First**. Jedná se o můj první pokus vytvořit plně responzivní webovou stránku.

## 🧩 Klíčové koncepty

</br>

| Koncept | Popis |
|---|---|
| 📱 **Mobile First** | Design navržen primárně pro mobilní zařízení, postupně rozšířen pro větší obrazovky |
| 📐 **Responzivní layout** | Přizpůsobení rozvržení všem velikostem obrazovky |
| 🎨 **Bootstrap grid** | Využití mřížkového systému pro flexibilní rozvržení |
